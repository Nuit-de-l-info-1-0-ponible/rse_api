# L'objectif de ce projet a été de :

- Creer une API qui permet de certifier ou non qu'un site web est éco-responsable selon ces performances et la quantité d'energie qu'il consorme mais pas que.
- Cette API intégre aussi des conseils
- Il permet aussi de directement tester le site en ligne
- L'objectif un de l'utiliser comme une bannière pour en sous couche comme pour les cookies
- On a utiliser nodes, express js
- On a aussi utiliser deux api existant déjà et open source à savoir : 'https://api.websitecarbon.com'; et 'https://api.thegreenwebfoundation.org'; ce qui donne un aspect plus performant au notre en plus il y'a cette aspect bannier qui agit comme un certificat et qui permet donc d'intéger la rse dès la conceptions c'est en ça que c'est une démarche RSE By Design
- Pour tester l'api vous pouvez utiliser l'appel au fichier test.html après avoir lancer l'api
